# Crypto Exchange

A modern cryptocurrency exchange platform built with React, TypeScript, and Vite.

## 🛠️ Installation

Clone the repository and install dependencies:

```sh
pnpm install
```

## 🔧 Scripts

Start the development server:

```sh
pnpm run dev
```

Build for production:

```sh
pnpm run build
```

Run tests:

```sh
pnpm run test
```

## 📦 Tech Stack

- React + Vite
- MobX
- Material UI
- Vitest
